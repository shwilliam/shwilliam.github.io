It seems like everyday people are discovering new ways to use the Instagram platform in creative ways to stand out. Unfortunately, the Instagram can be very limiting in many ways. For example, as of today, Instagram only exposes a way for users to add **one** working link per account, displayed under the user description. [This app](https://post-links.netlify.com/) allows IG users to quickly set up a page that they can redirect to, which mimics their account feed with working links to any URL specified in the post description (with `[link: https://some-link.com]`).
