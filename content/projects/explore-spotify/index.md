To explore the ways that D3 can be used with React, I created an interactive network diagram that allows users to explore Spotify recommendations, and export them as a playlist.

This project is a functional prototype and not fully developed. It has several bugs and cross-browser issues. I highly recommended that you view the project demo in the latest version of **Firefox**.
